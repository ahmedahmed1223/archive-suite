# إرشادات المساهمة

شكرًا لرغبتك بالمساهمة في مشروع "أرشيف الفيديو"! فيما يلي خطوات بسيطة لتسهيل التعاون:

1. افتح Issue لعرض فكرة جديدة أو خطأ تريد إصلاحه قبل البدء بالعمل (إن أمكن).
2. ارتكاب تغييرات صغيرة ومحددة، كل ميزة أو إصلاح يجب أن يكون في فرع منفصل.
   - نمط التسمية المقترح للفرع: `feature/<short-name>` أو `fix/<short-name>`.
3. أرسل Pull Request يصف التغيير والغرض وكيفية اختباره.
4. التزامات (commits): استخدم عبارات وصفية قصيرة ومباشرة.
5. التأكد محليًا أن التطبيق يعمل:

```bash
pnpm install
pnpm run dev
```

6. إذا قمت بتعديل الأنماط أو البنية، اذكر في الـ PR إذا كان مطلوبًا إعادة بناء (`pnpm run build:spa`).

نرحب بالمساهمات البسيطة: تحسين نصوص، إصلاح أخطاء صغيرة، توثيق، أو تحسينات في واجهة المستخدم.

إذا كنت تود المساهمة بشيء كبير، ابدأ بمناقشة الفكرة في Issue قبل كتابة الكود.

شكراً، ومرحباً بمساهمتك! ✨
