# عدسات الرسم وتنبيهات موجات الأعطال

## النطاق

تغلق الدفعة `V1-766` و`V1-735` دون API جديد أو اعتماد خارجي. تبقى البيانات محلية في المتصفح، وتستخدم الواجهة العربية وتوكنات التصميم الحالية.

## عدسات الرسم البياني

تعرض صفحة العلاقات عدسة «كل الأنواع» وعدسة لكل نوع موجود مع عدد عقده. اختيار العدسة يرشح العقد والصلات فورًا، ويحفظ الاختيار محليًا، ويعود تلقائيًا إلى «الكل» إذا لم يعد النوع المحفوظ موجودًا. تبقى فلاتر الوسم والبحث متوافقة مع العدسة.

## تنبيه موجة الأعطال

تُعد موجة مرتفعة عندما يبلغ مجموع مرات ظهور أخطاء severity=`error` خمس مرات أو أكثر خلال آخر خمس دقائق. يظهر تنبيه `role=alert` في صفحة الأخطاء يشرح العدد والنافذة، ولا يحتاج backend. يختفي تلقائيًا عندما ينخفض المعدل، ويعتمد على `count` و`lastSeenAt` الموجودين.

## الاختبارات والقبول

- اختبارات نقية لاستخراج العدسات واختيار العدسة الصالحة.
- اختبارات نقية لحساب معدل الأخطاء مع استبعاد التحذيرات والأحداث القديمة.
- اختبار واجهة لكل شريحة أساسية، ثم `pnpm test:next` و`pnpm build:next`.
- التزام مستقل لكل معرّف، ونقل البندين من `TASKS.md` إلى `ChangeLog.md`.
