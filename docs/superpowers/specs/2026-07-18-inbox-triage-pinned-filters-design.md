# الفرز السريع والفلاتر المثبتة

## V1-742 — فرز الوارد بلوحة المفاتيح

تضيف صفحة الوارد وضع فرز صريحًا. عند تفعيله يحتفظ بمؤشر على عنصر ظاهر واحد؛ `J/K` والأسهم تنقل المؤشر، والأرقام `1–4` تغيّر الحالة إلى جديد/قيد الفرز/جاهز/مكتمل، و`Enter` ينفذ الرابط الأساسي المتاح. لا تعمل الاختصارات داخل input/textarea/select أو عناصر contenteditable. تعرض الواجهة تلميح المفاتيح وتمييزًا واضحًا للعنصر النشط.

## V1-743 — الفلاتر المثبتة

تتيح صفحة الأرشيف حفظ مجموعة الفلاتر الحالية باسم مختصر لكل مستخدم. تظهر الفلاتر المثبتة كشريط قابل للتطبيق والحذف وإعادة الترتيب بأزرار صعود/هبوط قابلة للوصول؛ يُستخدم التخزين المحلي بمفتاح يتضمن معرف المستخدم. يشمل snapshot الاستعلام والنوع والوسم والحالة والعرض فقط، ويُهمل التخزين التالف.
