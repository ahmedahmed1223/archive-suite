# العلامات الزمنية والحضور على السجل

## V1-723

تُعامل العلامة الزمنية كملاحظة سجل مشتركة ذات `timestampSeconds`. يلتقط مشغل الوسائط الوقت الحالي، ويستدعي callback اختياريًا لإنشاء العلامة بعد إدخال وصف. تعرض صفحة السجل الملاحظات الزمنية مرتبة، والنقر على علامة يمرر وقتها إلى المشغل للقفز المباشر. لا يُنشأ جدول أو endpoint جديد.

## V1-724

يرسل مكوّن حضور السجل heartbeat إلى الغرفة `record:{id}` عند الفتح ثم دوريًا، ويعرض المشاركين الحاليين من الاستجابة. يفشل بصمت بصريًا برسالة صغيرة دون تعطيل تفاصيل السجل، ويتوقف المؤقت عند مغادرة الصفحة. الخادم يملك مهلة انتهاء الحضور.

## القبول

- اختبار تنسيق/ترتيب العلامات والتقاط وقت المشغل.
- اختبار مفتاح غرفة السجل وتحديث المشاركين عبر heartbeat.
- لا API أو migration جديد، ثم TypeScript واختبارات Next والبناء.
